WELCOME TO PMT

A Project Management Tool for Software Developers

What is it?

PMT is a project management tool primarily for software developers. It incoporates the functionality of a kanban board, which allows the user to organize their tasks as cards that can be moved to match specific project deadlines (sprints), and a chat system for each specific project.



Why use it?

PMT allows the user to organize their project tasks as cards that can be set to different priorities and assigned to different team members. These cards are automatically created in a Backlog and can be dragged into the necessary project sprints' In-Progress column. Once these tasks are completed, the user can move them into the sprint's Done column, allowing their team members to know that it has been completed.

PMT also allows the user to chat with their team members for each project. They can set up channels with specific titles; for example, there is a default channel called Design, this is were conversations about the project design can be put in order to keep the chat organized.
